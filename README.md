b64cp
====
Base64 encoding helper script.

### Installation
1. In windows right click on the script wherever you have it and select "create shortcut"
2. right click on the new shortcut and in the "Target" field enter "C:\path\to\pythonw.exe " before the current text
3. Click and drag the shortcut to the taskbar

### Usage
1. Highlight some text.
2. Click on the script shortcut
3. Paste your base64 string
